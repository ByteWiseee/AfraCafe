☕ Café Website Simulator
A modern and interactive café website built with HTML, CSS, and JavaScript that simulates the experience of a real coffee shop website. The project focuses on creating a clean user interface, smooth user experience, and practical features commonly used in modern café and restaurant websites.

🌟 Overview
This project is designed as a complete coffee shop website simulation, allowing users to explore the café, browse the menu, reserve tables, share feedback, and read blog content. The goal of this project is to demonstrate front‑end development skills while building a realistic web application structure.

✨ Features
🍽 Online Menu
Users can browse a beautifully designed menu showcasing the café’s food and drinks. The menu section provides a clear and organized view of available items and supports a smooth browsing experience.

🪑 Table Reservation System
Customers can reserve a table directly through the website by submitting their information. This simulates a real café reservation system and improves customer convenience.

💬 Customer Reviews
Visitors can leave feedback and share their experiences by submitting reviews with their personal details. This feature represents how cafés collect and display customer opinions.

📝 Blog Section
The website includes a blog area where articles related to coffee culture, café updates, or lifestyle topics can be shared with visitors.

🛒 Online Ordering Experience
The project includes a simulated online ordering section where users can explore menu items and experience a simple purchase flow.

ℹ About Us
An informative section introducing the café, its story, atmosphere, and mission to create a welcoming environment for coffee lovers.

🛠 Technologies Used
HTML5 – Structure and semantic layout
CSS3 – Styling, layout, and responsive design
JavaScript – Interactivity and client‑side functionality
🎯 Project Goal
The goal of this project is to practice building a realistic business website interface while improving skills in front‑end development, UI design, and interactive web components.
